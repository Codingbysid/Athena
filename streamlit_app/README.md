## 🚀 **Quick Start Guide**

### **1. Install Dependencies**
```bash
pip install streamlit plotly streamlit-option-menu streamlit-card
```

### **2. Run the Application**
```bash
cd streamlit_app
streamlit run 🏠_Home.py
```

### **3. Access the App**
```
Local URL: http://localhost:8501
Network URL: http://[your-ip]:8501
```
